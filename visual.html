Use Case: Visual Units

Beyond simple HTML, a FORWARD page can render visualizations using visual units, such as charts and maps. This example shows a visual unit that visualizes time series data using the popular HighCharts JavaScript library.

Example Code

To render a visual unit on index.page, the <% unit %> directive (lines 8-36) encloses the visual unit's data, such as the chart type, X-axis title and the time series for Phones sales. Data are represented with SQL++, which is an extension of JSON.

Key Feature

Notice that the visual unit automatically translates between SQL++ data and the underlying HighStock component. No JavaScript code is necessary to integrate the JavaScript component into the page!

The visual units documentation describes the SQL++ data attributes supported by respective visual units.

Exercises

  1. Add the SQL++ data for a Y-axis title. For example:
    yAxis : {
    title : {
        text : 'Total'
    }
},
  2. Add the SQL++ data for a second product. For example:
    {
    name : 'Tablet',
    data : [
        {
            // Epoch time for 2014-01-01
            x : 1388534400000,
            y : 401
        },
        {
            // Epoch time for 2014-01-02
            x : 1388620800000,
            y : 658
        },
        {
            // Epoch time for 2014-01-03
            x : 1388707200000,
            y : 740
        }
    ]
}
